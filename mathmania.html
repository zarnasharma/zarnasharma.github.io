Introduction

Math Mania

Summary

The brief given - Build a page with interactive and moving elements. Show off your knowledge of CSS3 and use transitions and animations; needs to be responsive, accessible.

Technologies

HTML5, CSS3, Vanilla JavaScript

The Process

Project Math Mania - Process

1

Research

For this project, my research was targeted at understanding how a boardgame cycle works. Also, I read up articles to understand CSS transitions, and animations on sites like CSSAnimationRocks, and practiced examples from the book, HTML5 and CSS3 Transition, Transformation, and Animation.

2

Storyboarding

Here, I drafted out various versions of the game, and how I envisioned the animations to fit in. On paper, everything seemed to gel in well, but the big question was - How would this work as a web app?

3

Phase Planning

I had 2 weeks to get this project planned, developed, styled, tested, and running. Having no prior experience in building a boardgame, I was looking forward to get coding, but I thought it was imperative to draft out at least two approaches to this project. So, I set up two goals to achieve at the end of this project. Achieving either one would fulfill the requirements set. Phase One was building the HTML skeleton for the boardgame, and Phase Two was JavaScript development.

4

Development

This was an amazing experience. The boardgame cycle to work as intented, which on paper looked simple, required a mathematical approach to it. I loved figuring this part out. Also, I was intrigued by how JavaScript and CSS3 animations could intertwine and give amazing effects - the game progress bar being an apt example. Responsiveness of the website was taken care by the simple grid system I use in all my projects. You can check it out here.

5

Tests & Tweaks

Although, testing wasn't done only at the end, but after every logical code milestone I achieved, this phase comprised of testing the overall game/user experience, and figuring out what could potentially go wrong. I learned that since arithmetic is not something that everyone loves, to ensure player engagement, the arithmetic questions level needs to be low to moderate.

The Outcome

Math Mania Screenshots

View Site

My first ever boardgame! I am extremely proud of this project as I accomplished to get it working without the aid of any fancy plugins whatsoever. I learned furthermore about how fascinating JavaScript is, and the power it holds. Currently, I intend to make this game a multiplayer one using Node.js and SocketIO with many more dynamic features like user accounts, difficulty levels, question categories, and the like.
